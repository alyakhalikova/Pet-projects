
# Тест-кейсы
## Тест кейс на позитивную проверку
**01 - Авторизация с валидными учетными данными**

**Предусловия:**  
Пользователь находится на странице входа [https://www.saucedemo.com/](https://www.saucedemo.com/). 
Интернет-соединение стабильно.  

**Шаги:**  
1. Перейти на сайт [https://www.saucedemo.com/](https://www.saucedemo.com/).  
2. Ввести логин `standard_user`.  
3. Ввести пароль `secret_sauce`.  
4. Нажать кнопку **Login**.  

**Ожидаемый результат:**  
Пользователь перенаправлен на страницу с товарами (`https://www.saucedemo.com/inventory.html`). Отображается список товаров. Ошибок и уведомлений нет.  

**Постусловия:**  
Пользователь авторизован в системе.

## Тест кейс на негативную проверку 
**02 - Проверка валидации формы входа при отсутствии пароля**

**Предусловия:**  
Пользователь находится на странице входа [https://www.saucedemo.com/](https://www.saucedemo.com/). Интернет-соединение стабильно.

**Шаги:**  
1. Перейти на сайт [https://www.saucedemo.com/](https://www.saucedemo.com/).  
2. Ввести логин `standard_user`.  
3. Оставить поле пароля пустым.  
4. Нажать кнопку **Login**.  

**Ожидаемый результат:**  
Отображается сообщение об ошибке:  
`Epic sadface: Password is required`.  
Пользователь остается на странице входа.  

**Постусловия:**  
Пользователь не авторизован. Ошибка отображена.
