# Тест-дизайн: Граничные значения для ползунка суммы займа


## 1. Граничные значения

 - Граница между 20% и 15% 
 - Граница между 15% и 10%
 - Минимальная и максимальная сумма


## 2. Тестовые значения

- `1 000`  (минимально допустимая сумма)
- `9 000`  (граничное значение перед переходом на 15% ставку)
- `10 000` (начало диапазона со ставкой 15%)
- `11 000` (внутри диапазона 15%)
- `49 000` (граничное значение перед переходом на 10% ставку)
- `50 000` (начало диапазона со ставкой 10%)
- `51 000` (внутри диапазона 10%)
- `100 000` (максимально возможная сумма займа)
