# Тест-дизайн: Класс эквивалентности
## 1. Классы эквивалентности

- Латинские буквы 
- Цифры
- Специальные символы, знаки пунктуации
- Пробелы 
- Символы кириллицы 
- Символы латиницы
- Комбинация цифр, букв и специальных символов


## 2. Тестовые данные для каждого класса эквивалентности

- `abcXYZ` (латинские буквы)
- `1234567890` (цифры)
- `!@#%&_+,.` (специальные символы)
- `login test` (с пробелами)
- `тестовыйлогин` (кириллица)
- `john` (английский язык)
- `john123!-+` (комбинация)
