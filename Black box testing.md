# Чек-лист Black-box тестирования по техническому заданию


## SC21-01 – Поле ввода названия магазина
- [ ] Поле отображается на экране.
- [ ] В поле можно ввести любые символы (буквы, цифры, спецсимволы).
- [ ] В поле отображается плейсхолдер.
- [ ] Плейсхолдер исчезает при вводе символов.
- [ ] При вводе 1 символа — кнопка «Поиск» не активна.
- [ ] При вводе 2 символов и более — поле считается валидным.
- [ ] При повторном открытии поля отображаются предыдущие поисковые запросы.

## SC21-02 – Поле выбора типа магазина
- [ ] Поле выбора отображается на экране.
- [ ] По умолчанию в поле отображается плейсхолдер «Выберите тип магазина».
- [ ] При нажатии открывается выпадающий список.
- [ ] Список содержит варианты:  
  - «Продуктовые магазины»  
  - «Магазины стройматериалов»  
  - «Аптеки»
- [ ] После выбора значение отображается в поле.

## SC21-03 – Кнопка «Поиск»
- [ ] Кнопка «Поиск» отображается на экране.
- [ ] Кнопка неактивна, если оба поля пусты.
- [ ] Кнопка активна, если заполнено хотя бы одно поле.
- [ ] При нажатии на активную кнопку отображаются флажки на карте.
- [ ] Карту можно скроллить и масштабировать стандартными жестами (двумя пальцами / колесиком мыши).

## SC21-04 – Кнопка «Построить маршрут»
- [ ] Кнопка отображается с текстом «Построить маршрут» и белым фоном (#fff).
- [ ] Кнопка содержит иконку в виде изогнутой стрелки.
- [ ] Кнопка становится активной при:
  - Поиске магазина и выборе флажка на карте.
  - Выборе типа магазина.
- [ ] При нажатии на кнопку:
  - Фон становится серым (#586d73).
  - Текст меняется на «Отменить маршрут».

## SC21-05 – Область карты
- [ ] Карта занимает половину экрана телефона.
- [ ] Построение маршрута возможно только при выполнении условий SC21-04.
- [ ] Построенный маршрут можно отменить только кнопкой «Отмена».

## SC21-06 – Общие требования
- [ ] При повторном открытии приложения отображается незавершённый ранее маршрут.
- [ ] При низком заряде аккумулятора отключается точная геопозиция.
